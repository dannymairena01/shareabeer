# @sab/mobile

Expo React Native app for Share a Beer. Phase 0 ships a splash screen;
Phase 1 adds auth + age-verification + profile (UC-1, UC-21).

## Run

```bash
pnpm --filter @sab/mobile start       # Expo dev server
pnpm --filter @sab/mobile ios         # iOS simulator (requires Xcode)
pnpm --filter @sab/mobile android     # Android emulator (requires Android Studio)
```

## Verify bundling

```bash
cd apps/mobile && pnpm exec expo export --platform ios --output-dir /tmp/sab-ios-export
```

Exits 0 and writes a Hermes bundle; a clean run is the Phase 0 gate for
the mobile app. Also runnable for `--platform android`.

## Notes

- Expo SDK **54** (React Native 0.81.5, React 19.1). See
  `docs/adr/0007-expo-sdk-54.md` for why we are one SDK behind current.
- Monorepo-aware Metro config in `metro.config.js` hot-reloads edits in
  `@sab/shared-types` and `@sab/shared-validation`.
- The repo uses pnpm's **hoisted** node-linker (see `/.npmrc` and ADR
  0006). Do not switch to the isolated linker without reading that ADR —
  Metro's resolver breaks on pnpm symlinks for transitive deps.
- Sentry DSN is loaded from `EXPO_PUBLIC_SENTRY_DSN`; leave empty in dev.
- iOS / Android folders are generated by `expo prebuild` and are gitignored.
